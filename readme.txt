"Login Branding Pro" is a WordPress plugin that allows you to easily customize the login page of your website with your branding elements.

== Description ==
The "Login Branding Pro" plugin provides a simple and intuitive way to personalize the WordPress login page to match your website's branding. With this plugin, you can customize the background color, upload a custom logo, and change the link associated with the logo. Give your users a seamless login experience that reflects your website's identity.

==Features==

- Customize the background color of the login page to match your brand's color scheme.
- Upload a custom logo to replace the default WordPress logo on the login page.
- Change the link associated with the logo to redirect users to a specific page.
- Simple and user-friendly settings interface for easy customization.
- Enhance the login experience and maintain a consistent brand presence throughout your website.

==Installation==

1. Download the plugin zip file
2. Upload the plugin directory to the /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the plugin settings page to customize the login page
